feature
======

1. Generate features   

    * Data format the same as Blei's LDA-C 
        * http://www.cs.princeton.edu/~blei/lda-c
        * One document per line
        * Each line has three fields separated by tab
            * [doc-id] [doc-category] [doc-body]
    * Three sets of features          
        * unigram only
        * bigram only
        * unigram + bigram
