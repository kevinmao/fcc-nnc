train
======

1. Import

    * Import and convert data into mallet format
    
2. Split

    * Split data into 80% training and 20% testing

3. Train

    * Train three sets of features on 80% data         
        * unigram only
        * bigram only
        * unigram + bigram
        
4. Evaluate

    * Evaluate three sets of features on 20% data         
        * unigram only
        * bigram only
        * unigram + bigram
